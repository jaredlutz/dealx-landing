# DealX Landing

Institutional fixed income landing page built with Next.js and deployed on Vercel.

## Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Animation:** Framer Motion
- **Icons:** Lucide React

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy to Vercel

1. Push this repo to GitHub (e.g. `jaredlutz/dealx-landing`)
2. Go to [vercel.com](https://vercel.com) → Import Project
3. Select the repo and deploy (zero config required)

Or use the Vercel CLI:

```bash
npm i -g vercel
vercel
```

## Structure

- `app/` — Next.js App Router (layout, page, globals)
- `components/` — Landing page sections and UI components
